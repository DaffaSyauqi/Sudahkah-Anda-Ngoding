# Sudahkah Anda Ngoding Hari Ini?

Aplikasi web sederhana yang ringan, interaktif, dan menampilkan statistik kebiasaan ngoding dalam bentuk visualisasi grafik.
Jadi apakah anda sudah ngoding hari ini???

---

## Fitur Utama
- Penyimpanan data berbasis file JSON
- Visualisasi statistik
- Tampilan sederhana

---

## Tech Stack
- [Python](https://www.python.org/)
- [Matplotlib](https://matplotlib.org/)
- [Streamlit](https://streamlit.io/)
- JSON (File-based storage)

---

## Instalasi

### 1. Clone Repository
```bash
git clone https://github.com/DaffaSyauqi/Sudahkah-Anda-Ngoding.git
cd sudahkah-anda-ngoding
```

### 2. Install Dependency
```bash
pip install -r requirements.txt
```

### 3. Running Local
```bash
streamlit run app.py
```
